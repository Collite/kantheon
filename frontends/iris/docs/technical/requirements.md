# Requirements for New Frontends

This document outlines the requirements for developing new frontend for the project.

## The Goal

The goal is to create a new frontend, copying a part of the existing aip-fe frontend:
The frontend will be named "agents-fe", a frontend exposing the communication with the agents

### Agents
Agents will be mostly a copy of the "Agent", "LLM Gateway" and "MCP Inspector" views of the existing aip-fe frontend.
The only enhancement will be that the Agent view should allow the used to select the agent to talk to:
- there will be a submenu in the left hand side menu "Agents" -> "ERP Agent"
- the "ERP Agent" is the existing "Agent" view

All the other functionality, configurations etc will be copied from the existing aip-fe frontend.
