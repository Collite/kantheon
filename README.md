# Kantheon

The agent constellation that sits on top of [`ai-platform`](../ai-platform). Kantheon owns
the user-facing surface (Iris), the question understanding + routing agent (Themis),
the autonomous investigator (Pythia), the per-domain Q&A template (Golem), and the
unified agent + tool registry (`tools/capabilities-mcp`).
